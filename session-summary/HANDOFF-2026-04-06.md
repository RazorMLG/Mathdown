## Key Decisions

| Decision             | Choice                                                                     | Reason                                                                                                                                                |
| -------------------- | -------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| Color system         | CSS custom properties (`:root` / `.dark`)                                  | Allows instant theme switching without re-rendering the component tree; both Tailwind utility classes and raw inline styles can consume the same vars |
| Theme switching      | `.dark` class on root div + `useState`                                     | Tailwind v4 dark mode is class-based; state in App.tsx drives it cleanly                                                                              |
| Initial theme        | `window.matchMedia('(prefers-color-scheme: dark)')`                        | Respects OS preference on first load; user can override with toggle                                                                                   |
| Toolbar placement    | Fragment returned from `Toolbar.tsx`, rendered inside Sidebar header `div` | Keeps component boundary intact while achieving the "icons in header" layout                                                                          |
| Panel labels         | Absolute-positioned 10px floating text                                     | Notion-style — no chrome, no border, just a subtle hint                                                                                               |
| Accent color         | Indigo (`#4f46e5` light / `#818cf8` dark)                                  | User chose indigo/violet explicitly                                                                                                                   |
| Sidebar width        | 240px (was 250px)                                                          | Slightly tighter for cleaner proportions                                                                                                              |
| Note content preview | Strip `#`, `**`, `*`, `` ` `` via regex; slice to 120 chars                | Shows enough to identify the note without implementing a full text extractor                                                                          |
